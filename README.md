# ACC102-BYD-Data-Analysis
## Project Overview
This project analyzes BYD's stock price (A-share & HK-share) trends and monthly sales volume using Python. It aims to find the relationship between business performance and capital market performance.

## Dataset
1. Stock Price: Yahoo Finance, 2025-01-01 to 2026-04-01
2. Sales Data: BYD official monthly sales announcements
3. Access Date: 2026-04-09

## Tools & Libraries
Python, Pandas, Matplotlib, yfinance

## Workflow
Data Acquisition → Data Cleaning → Descriptive Analysis → Visualization → Insight Generation

## Key Insights
1. BYD A-share and HK-share have highly consistent price trends.
2. Strong sales growth in 2025 provided fundamental support for the stock.
3. Sales performance and stock price show a positive correlation in most periods.

## How to Run
1. Install dependencies: pip install pandas yfinance matplotlib
2. Open byd_analysis.ipynb in Jupyter Notebook
3. Run all cells to view results
