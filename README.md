# ACC102-BYD-Data-Analysis
## Project Overview
This project analyzes BYD's stock price (A-share & HK-share) trends and monthly new energy vehicle sales using Python. It explores the relationship between operational performance and capital market performance for business and financial analysis.

## Dataset
1. Stock Price: AKShare (2025-01-01 to 2026-04-01)
   - A-share: 002594.SZ
   - HK-share: 1211.HK
2. Sales Data: BYD Official Monthly Announcements
3. Access Date: 2026-04-09

## Tools & Libraries
Python, Pandas, Matplotlib, AKShare

## Workflow
Data Acquisition → Data Cleaning → Descriptive Analysis → Visualization → Insight Generation

## Key Insights
1. BYD A-share and HK-share have highly consistent price trends.
2. Strong sales growth in 2025 supports stock performance.
3. Sales volume shows obvious seasonality in Q1 2026.

## How to Run (Mac)
1. Install dependencies: pip3 install pandas akshare matplotlib notebook
2. Run byd_analysis.ipynb in Jupyter Notebook
3. All cleaned data will be saved as CSV files automatically
